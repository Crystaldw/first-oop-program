Пул строк в Java:

В Java "пул строк" (string pool) - это механизм, который используется для управления строковыми литералами. Когда вы создаете строковый литерал в Java (например, String str = "Hello";), JVM автоматически проверяет, есть ли уже строка с таким значением в пуле строк. Если строка с таким значением уже существует, JVM использует существующий объект String, а не создает новый.

String str1 = "Hello"; // Создается строка в пуле
String str2 = "Hello"; // Используется существующая строка из пула
String str3 = new String("Hello"); // Создается новый объект String (не в пуле)

System.out.println(str1 == str2); // Выведет true, так как обе переменные указывают на один и тот же объект в пуле строк
System.out.println(str1 == str3); // Выведет false, так как это два разных объекта String
